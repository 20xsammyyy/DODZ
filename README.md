# DODZ

DODZ is a retro-style arcade dodger game inspired by classic 1980s pixel games.

The goal of the game is simple: survive as long as possible while avoiding enemies falling from the top of the screen.

## Gameplay

- Move left and right to dodge falling enemies.
- The longer you survive, the higher your score.
- If an enemy hits the player, the game ends.

## Controls

Left Arrow  ←  : Move Left  
Right Arrow →  : Move Right  
Spacebar        : Start / Restart Game

## Features

- Retro pixel-style visuals
- Arcade-inspired gameplay
- Score tracking
- Simple keyboard controls
